الأب يعتمد على وكلاء ذكاء اصطناعي وبيانات خيالية (Nova Bay). متطلبات Baseera تمنع إرسال بيانات حساسة لخدمات AI خارجية وتمنع اتخاذ قرارات أمنية آلية. لذلك **لا يُعاد استخدام** كود الأب في هذه المنصة.
