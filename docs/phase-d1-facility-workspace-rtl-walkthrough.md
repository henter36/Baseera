# Phase D.1 Facility Workspace RTL Walkthrough

- Page uses `WorkspaceShell` with `dir="rtl"`.
- Header, filter bar, widgets, badges, links, and timeline inherit existing RTL tokens.
- Desktop uses multi-column workspace grid.
- Tablet uses shared grid collapse behavior.
- Mobile uses single column, full-width filters, and no horizontal scrolling.
- Drill-down links are real anchors styled as buttons for keyboard and screen-reader consistency.
