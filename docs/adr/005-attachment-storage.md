# ADR-005: تخزين المرفقات عبر واجهة قابلة للاستبدال

## الحالة

مقبول — المرحلة A

## السياق

المرفقات تتطلب تحقق نوع/حجم/بصمة وتدقيق تنزيل، مع إمكانية الانتقال لاحقًا لتخزين سحابي مؤسسي.

## القرار

واجهة `IFileStorage` مع تنفيذ محلي (`LocalFileStorage`) للتطوير. البيانات الوصفية في SQL؛ المحتوى على القرص تحت مجلد مُعد. فحص أمني عميق قابل للتكامل لاحقًا عبر حالة `PendingScan`.

## العواقب

- لا ارتباط ببائع تخزين في المرحلة A.
- يجب تأمين مسار التخزين المحلي في النشر.
