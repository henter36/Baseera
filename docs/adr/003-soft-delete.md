# ADR-003: Soft Delete بدل الحذف النهائي

## الحالة

مقبول — المرحلة A

## السياق

السجلات التشغيلية والتنظيمية حساسة وتتطلب إمكانية تدقيق وأرشفة.

## القرار

استخدام Soft Delete (`IsDeleted`, `DeletedAtUtc`) للكيانات التشغيلية والتنظيمية. منع hard delete من طبقة التطبيق. AuditLog غير قابل للحذف أو التعديل.

## العواقب

- استعلامات EF مع Global Query Filter.
- حاجة لسياسات احتفاظ وأرشفة لاحقة.
