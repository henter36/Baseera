# ADR-001: Modular Monolith

## الحالة

مقبول — المرحلة A

## السياق

منصة واسعة النطاق (تنظيم، صلاحيات، ملاحظات، نماذج، موارد، استراتيجية). Microservices مبكرًا يزيد تعقيد التشغيل دون حاجة تشغيلية مثبتة.

## القرار

بناء Modular Monolith بطبقات Domain / Application / Infrastructure / Api، مع وحدات وظيفية كمجلدات داخل الطبقات، ومشاريع مساعدة Reporting و BackgroundJobs.

## العواقب

- نشر واحد، معاملات أسهل، اختبارات تكامل أبسط.
- يجب الحفاظ على حدود وحدات واضحة لتسهيل الاستخراج لاحقًا إن لزم.
