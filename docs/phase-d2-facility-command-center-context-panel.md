# Phase D.2 Facility Command Center Context Panel

The context panel keeps the user inside the facility workspace while inspecting operational items.

Capabilities:
- Opens from priority rows and recent activity.
- Supports direct links via URL query state.
- Lazy-loads details only for the active entity.
- Shows full-page route as a secondary action.
- Supports close, Escape, and focus return.

Entity support:
- Notes: summary, status rail, description, corrective actions, timeline, supported inline actions.
- Corrective actions: status, priority, due date, assignment, history.
- Escalations: priority preview and safe operational reason.
- Forms: compliance preview and explicit full-page route for entry/review.
- Activity: event reference, description, timestamp, tone.

Deferred:
- Full escalation acknowledgement inside the panel.
- Full corrective action assignment and verification forms inside the panel.
- Full form response entry inside the panel.
