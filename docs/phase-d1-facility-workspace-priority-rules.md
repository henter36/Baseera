# Phase D.1 Facility Workspace Priority Rules

The queue is deterministic and bounded to 10 final items.

Sources:
- critical open notes: base rank 90.
- overdue notes: base rank 80 plus capped overdue days.
- active escalations: base rank 75 plus escalation level.
- overdue corrective actions: base rank 70 plus capped overdue days.
- overdue forms: base rank 65 plus capped overdue days.

Ordering:
1. Higher rank.
2. Earlier due date.
3. Stable reference ordering.

The queue is not a Risk Score and does not use AI.
